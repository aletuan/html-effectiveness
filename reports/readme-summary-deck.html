What's in the repo

A gallery, not a framework

Every file in this repo is a self-contained .html page. Open one in a browser and it just works — useful as reading material, useful as a styling reference, useful as a target format for tools that need to produce a real artifact.

Start at index.html for the categorized index, or open any numbered file directly.

Sample code · not maintained · not accepting contributions.

Running

Nothing to install

  • 01
    Clone the repo
    No package manager, no toolchain. The files in the repo are the artifacts.
  • 02
    Open index.html
    The categorized index links to every example. Or jump straight to a numbered file like 12-incident-report.html.
  • 03
    That's it
    Each page opens directly in a browser — no server, no network, no fonts to load.
Also ships as

A Claude Code skill

The gallery doubles as a reusable output format for Claude Code.

Ask Claude to "review this PR and export to HTML", "draft an incident postmortem", or "write up a status report" — the skill picks the matching template and produces a single self-contained .html file styled like the rest of the gallery.

cp -r .claude/skills/html-output ~/.claude/skills/
