Dark mode palette — candidate values

A warm dark counterpart to the gallery's ivory/clay/oat system. Token names stay identical; only the values change. Surfaces are warm near-black (not pure black), text is warm off-white (not pure white), accents keep their hue but are lifted ~12% for contrast.

This file is itself rendered with the candidate tokens and declares <meta name="color-scheme" content="dark"> — the same self-declaration convention a generated dark artifact would carry. Review the swatches, then the "in context" card below to judge the colors working together.

Add backpressure guard to the streaming writer

The writer now stops pulling from the upstream queue once the socket's high-water mark is reached, instead of buffering unboundedly. Body text sits at --gray-700 on a --paper card; the hairline border is the lifted --gray-300.
